# DESIGN.md - 即使闯关大师

## 项目定位
面向 HSK4 级以上中高级汉语学习者的"即使……也……"让步复句闯关教学游戏网页。

## 气质与意象
- **核心意象**：闯关冒险 + 中式书卷气
- **场景叙述**：在古色古香的书院中，学习者化身侠客，通过一道道语法关卡，最终掌握"即使"的奥义，获得"闯关大师"称号
- **整体感觉**：亲切活泼又不失文化底蕴，学习过程像一场有趣的探险

## 视觉策略
- **图形语言**：水墨风渐变背景 + 现代卡片式布局，传统与现代融合
- **主视觉元素**：卷轴、印章、毛笔字风格的标题
- **关卡进度**：用阶梯/山路图标体现闯关感

## 配色方案
| 用途 | 色值 | 意象来源 |
|------|------|---------|
| 主色（墨玉青） | #0F766E | 中国传统墨玉色，沉稳有书卷气 |
| 辅助色（朱砂红） | #DC2626 | 印章朱红色，强调与警示 |
| 成功色（竹青） | #16A34A | 翠竹之色，答对与进阶 |
| 背景色（宣纸米白） | #FEFCE8 | 宣纸底色，温暖不刺眼 |
| 文字主色 | #1F2937 | 深灰近墨，正文易读 |
| 文字辅色 | #6B7280 | 中灰，提示与辅助信息 |

## 字体排版
- **字体族**：系统中文 + "Noto Serif SC"（思源宋体，营造书卷气）
- **标题**：粗体、大字，配毛笔字风格装饰
- **题目**：中等字号，行间距宽松，方便阅读
- **选项卡片**：清晰层级，悬停有反馈

## 动效与交互
- **过渡曲线**：cubic-bezier(0.4, 0, 0.2, 1)，自然流畅
- **答题反馈**：正确→绿色闪光 + 轻微弹跳；错误→红色轻抖 + 解析淡入
- **关卡切换**：淡入淡出 + 轻微上移
- **进度条**：平滑填充动画

## 设计禁忌
- 不要使用太花哨的卡通风格，保持一定的文化底蕴
- 不要使用纯黑背景，用暖米白营造护眼的学习氛围
- 题目文字不要太小，确保中高级学习者阅读舒适
- 不要跳过反馈步骤，每次答题后都要有明确解析
